# Trivia-Desktop

Trivia desktop is a fun and simple desktop app for playing trivia!

# Installation

Download the installer for windows [here](https://github.com/gubareve/trivia-desktop/releases/download/v1.2.0/trivia-desktop-setup-1.2.0.exe) and the installer for mac [here](https://github.com/gubareve/trivia-desktop/releases/download/v1.2.0/trivia-desktop-1.2.0.dmg).
